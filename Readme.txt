To run, execute nodemon index.js from employee-feedback-backed dir
                ng serve from employee-feedback dir

Database used : mongodb;
Backend :Nodejs with express
Frontend : Angular with bootsrap, css


There is an admin login at localhost:4200/admin/login.
There is a employee login at localhost:4200/employee/employee-login.

Admin Credentials : username = admin@gmail.com, password = Admin@123

Employee Credentials have to be entered by the admin when adding an employee.

To assign an employee for review, a reviewer and employee must be selected and performance review must be assigned to pending.
